# Next.js App with User Auth

## Goal

A Next.js App where a user can register / login and have role dependent access rights.
The db will be based on a postgres db provided by Supbabase.

### Dependencies / Stack

- Next.js
- Supabase
- Lucia
- Drizzle
- Shadcn

# Features
- SignIn / SignUp
- Passwort forget / reset
- Email verification
- 2-factor support
- Delete Account
- Send Emails
